# docker-btcd

This is a Docker-compose file for Btcd.

## Quickstart

1. Install [Docker](https://docker.com/)
2. Git clone [btcd](https://github.com/SebastianElvis/btcd)
3. From the cloned directory, build and run

```bash
docker-compose up
```
